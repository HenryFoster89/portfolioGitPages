## GitHub Pages Setup

To deploy this portfolio:

1. **Initialize Git repository**
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio setup: homepage with 3 project showcases"
   ```

2. **Create GitHub repository**
   - Repository name: `portfolioGitPages` (or `[username].github.io` for user site)
   - Public repository
   - No README initialization (we're creating our own)

3. **Push to GitHub**
   ```bash
   git remote add origin https://github.com/[YOUR_GITHUB_USERNAME]/portfolioGitPages.git
   git branch -M main
   git push -u origin main
   ```

4. **Enable GitHub Pages**
   - Go to repository Settings → Pages
   - Source: Deploy from branch `main`, folder `/ (root)`
   - Save and wait 1-2 minutes for deployment

5. **Access your site**
   - URL: `https://[YOUR_GITHUB_USERNAME].github.io/portfolioGitPages/`
